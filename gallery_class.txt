1. 인터셉터라는 기능을 이용하여 클라이트의 권한 기능 추가

   - 백엔드 : spring framework interceptor
     로그인하지 않은 사용자의 접근을 허용하지 않겠다.
     로그인하지 않은 사용자가 특정 API 접근하려고 하면
     사전에 차단하는 기능을 추가 하겠다.

      interceptor  : 중간에 가로채기 기술
      spring interceptor  : 클라이언트가 요청한 HTTP Request 를 가로챈다.

    로그인하지 않은 사용자가 접속했다면
    로그인 회원 아이디가 없으면 응답 객체의 상태 코드 401(Unauthorized : 권한 없음)
    입력하고, false 리턴 처리 ,
    이 렇게 인터셉터 단에서 처리하게 되면
    클라이언트의 요청이 컨트롤로러 전달 되 지 않고 차단된다.
